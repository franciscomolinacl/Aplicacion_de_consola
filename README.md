# Aplicación de consola

Proyecto del Módulo 3: Aplicación de consola.

## Descripción

Esta aplicación permite realizar las siguientes operaciones matemáticas:

- Sumar, restar y multiplicar x números.
- Operaciones con dos números.
    - Dividir.
    - Potencias.
    - Raiz.
    - Módulo.
    - Porcentaje (3 opciones).
    - Ley Ohm y Watt (Potencia, Amperes, Voltaje y Ohm).
    - Números correlativos desde x número hasta x cantidad de números.
    - Fibonacci desde x número hasta x cantidad de números.

## Tecnologías usadas

- HTML
- CSS
- JavaScript
- Bootstrap

## Conceptos aplicados

- Variables.
- Condicionales.
- Bucles.
- Funciones.
- Arreglos.
- Objetos.

## Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en el navegador.
3. Abrir la consola del navegador.
